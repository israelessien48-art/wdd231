Individual Project Site Plan

Site Name

Naija Student Tech Hub

This name represents a centralized online hub focused on helping Nigerian students learn, practice, and grow skills in web development and technology. The word Naija connects the site to Nigeria, while Tech Hub reflects learning, collaboration, and innovation.

Site Purpose

The purpose of this website is to provide learning resources, project examples, and guidance for students studying web development and computer science. The site will include tutorials, project ideas, best practices, and links to useful tools to support student success.

Scenarios

  • How can I learn HTML, CSS, and JavaScript as a beginner?
  • Where can I find simple project ideas to practice my web development skills?

Color Scheme

  • Primary Color: #1f4fd8 – Headings and emphasis
  • Secondary Color: #f2f4f8 – Page background
  • Accent Color: #111827 – Body text

Typography

  • Georgia, serif – Headings
  • Arial, sans-serif – Body text
